O BI é composto basicamento por:

Ferramentas, 
Infraestrutura,
Profissionais(corpo técnico),
Dados. 

                        BI DATA 
De onde vem os dados?

Dados gerencias -> BI Data

Pesquisa de campo -> Data 
Indicadores de mercado -> BI Data 

                        BI SOLUTION

Infraestrutura ->> gerenciamento dos dados ->> Analytcs ->> Compartilhamento ->>
Ferramentas gerais.
